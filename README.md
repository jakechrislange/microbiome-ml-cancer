# Differentiating between Cancer Types with Microbiome-Based Machine Learning Models

This repository was used for a machine learning project with the goal of classifying cancer types from TCGA using microbiome data from TCMA. 

This started as a project for a graduate course within the University of Texas at Austin's MSCSO program. See the PDF Differentiating_COAD_READ_Microbiome_ML.pdf for the paper produced for the class.
